# 小程序高保真流程原型 v2

入口文件：`index.html`

本版本保留旧版原型目录 `prototypes/wxapp-hifi/` 不变，用 `prototypes/wxapp-hifi-v2/` 做新版对照。

## v2 优化重点

- 首页首屏优先表达“找货、清库存、找厂”，减少泛运营信息对搜索入口的干扰。
- 搜索结果卡强化快速判断：认证、平台核实、刷新时间、数量、价格和商家活跃。
- 搜索无结果承接采购需求，减少用户重复录入。
- 资源详情强化信任证据：实拍图、核实标签、商家信用、最近活跃和联系前提示。
- 商家工作台区分买家和商家路径，突出权益、发布效果和下一步建议。
- 我的发布从单纯状态管理升级为“数据 + 建议 + 操作”的资源运营面板。
- 消息中心按待处理和业务类型组织，突出审核、过期、撮合和效果反馈。

## 验证目标

- 买家能在 5 秒内理解平台解决的是找货、清库存和找厂。
- 买家能从首页搜索进入资源详情并完成联系判断。
- 买家搜索无结果时能自然提交采购需求。
- 商家能理解发布资源、认证权益、置顶券和发布效果。
- 商家能根据效果数据完成刷新、置顶、标记成交、再发类似。
- 消息中心能承接审核、过期、撮合和效果反馈，并给出明确下一步动作。
