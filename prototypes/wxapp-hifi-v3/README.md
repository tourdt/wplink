# 小程序高保真流程原型 v3 - 上线标准版

入口文件：`index.html`

本版本保留旧版原型 `prototypes/wxapp-hifi/` 和 v2 原型 `prototypes/wxapp-hifi-v2/` 不变，用 `prototypes/wxapp-hifi-v3/` 作为上线前评审版本。

## 上线标准

- 首屏保留旧版高保真视觉焦点，用户能快速理解找货、清库存、找厂。
- 搜索结果支持快速判断：平台核实、认证状态、刷新时间、数量、价格、联系热度。
- 搜索无结果可自然转为采购需求，覆盖需求提交和消息通知。
- 资源详情展示实拍、规格、商家信用、联系前提示和同类推荐。
- 商家可发布资源、保存状态、提交审核，并理解曝光、联系和置顶权益。
- 我的发布覆盖已发布、待审核、即将过期、已过期等上线必须状态。
- 消息中心覆盖审核、过期、需求跟进和效果反馈。
- Banner、专题和 web-view 覆盖上线验收所需的跳转和白名单说明。
